Zookeeper Admin 简介

Hadoop、Storm、Kafka、Hbase等大数据集群组件使用越来越多，依赖Zookeeper进行工作，自然而然Zookeeper的维护工作尤为重要！！

Zookeeper在大数据生态圈中举足轻重，而，针对Zookeeper集群的监控工具相对较少或者监控比较片面。

基于此，Zookeeper Admin应运而生！！


重点功能特性

  • 集群配置  支持多Zookeeper集群配置，动态增减监控集群
  • 节点信息  方便查询znode的 meta信息、acl信息及data值
  • 集群状态  集群状态一目了然，可视化显示集群中各服务器的实时状态，以及事物处理量等
  • 监控方面  实时监控预警，针对集群，可自定义设置监控项目，配置预警级别（微信、短信、邮件、电话）
